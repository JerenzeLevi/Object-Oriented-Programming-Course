# 📖 Overview

The Prefinal phase focused on the concept of **Inheritance**, one of the core principles of Object-Oriented Programming.

Inheritance allows classes to acquire properties and behaviors from other classes, improving code reusability and structure.

---

# 🧠 Topics Covered

* 🧬 Inheritance
* 👨‍👩‍👧 Parent & Child Classes
* ♻️ Code Reusability
* ⚙️ Extending Existing Classes
* 🧩 Method Sharing
