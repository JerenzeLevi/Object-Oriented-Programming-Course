# 📖 Activity Overview

This activity demonstrates the basic implementation of **Object-Oriented Programming (OOP)** concepts using a simple WPF application in C#.

The project focuses on:

* Creating a custom class
* Defining attributes
* Creating methods
* Instantiating objects
* Triggering object behavior through UI interaction

---

# 📝 Instructions

Create a class named `CarClass` with the following attributes:

* 🚗 `model`
* 🏷️ `brand`
* 🎨 `color`
* ⚖️ `weight`

Inside the class, define a method named:

```csharp id="q4z7we"
accelerate()
```

The method should display a message box containing:

```text id="s8m1xf"
"The car is accelerating"
```

---

# 🖱️ User Interaction

Inside `MainWindow.xaml.cs`:

* Instantiate the `CarClass`
* Create a button labeled **Accelerate**
* Call the `accelerate()` method inside the button click event

When the button is clicked:

* 💬 A message box appears showing the acceleration message.
