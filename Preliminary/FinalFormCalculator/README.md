# 📖 Overview

This project is an enhanced and redesigned version of the basic calculator activity developed using C# and Windows Forms.

Unlike the standard calculator layout, this version was creatively redesigned into a:

# 🍓 Strawberry-Themed Calculator

The application features:

* Soft pink-inspired interface
* Red-toned typography
* Circular button designs
* Improved UI aesthetics
* Enhanced user interaction

The project combines both:

* 💻 Functionality
* 🎨 Creativity in interface design

# 🎨 UI Design Concept

The calculator was styled using:

* Pink color palettes
* Red accent typography
* Rounded/circular controls
* Soft aesthetic interface elements

The goal was to create a calculator that feels:

* More visually engaging
