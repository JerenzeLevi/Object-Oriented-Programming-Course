# 📖 Overview

This project is a simple calculator application developed using C# and Windows Forms as part of the introductory activities in Object-Oriented Programming.

The application performs basic arithmetic operations while demonstrating:

* Event-driven programming
* GUI development
* Basic programming logic
* User interaction handling

---

# ✨ Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🖱️ Button-based interaction
* 💻 Simple graphical user interface
