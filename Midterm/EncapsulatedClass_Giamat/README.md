# 📖 Overview

This project demonstrates the principle of **Encapsulation** in Object-Oriented Programming using C# and Windows Forms.

The application allows users to enter personal information through a graphical interface while securely managing data using:

* 🔒 Getters
* 🔒 Setters
* 🧩 Encapsulated attributes

The design of the application follows a soft and tiny-cutesie aesthetic featuring calming blue-to-indigo tones with a minimalist interface style.

---

# 📝 Activity Instruction

Create a C# program that asks the user to provide five pieces of personal information.

Design a class where:

* All attributes are encapsulated
* Information is accessed securely using getters and setters

Inside the main form:

* Add textboxes for user input
* Add a button that displays all entered information in a MessageBox

The application should demonstrate:

* Proper encapsulation
* Organized program structure
* User-friendly interaction
