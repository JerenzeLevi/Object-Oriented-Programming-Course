# 📖 Overview

The Midterm phase focused on understanding and applying the principle of **Encapsulation** in Object-Oriented Programming.

Encapsulation allows data and behaviors to be organized securely inside classes while controlling access using modifiers and methods.

---

# 🧠 Topics Covered

* 🔒 Encapsulation
* 🧩 Classes & Objects
* 🛡️ Access Modifiers
* ⚙️ Methods and Behaviors
* 📦 Data Organization

# 💻 Activities

This folder contains exercises and projects demonstrating:
