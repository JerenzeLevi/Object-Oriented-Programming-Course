# 📖 Overview

This project demonstrates the concept of **Encapsulation** in Object-Oriented Programming using C# and Windows Forms.

The application allows users to input personal information through a graphical user interface and securely store the data inside a class using:

* 🔒 Getters
* 🔒 Setters
* 🧩 Encapsulated attributes

The program was designed to be:

* User-friendly
* Organized
* Simple yet visually clean

---

# 📝 Activity Instruction

Create a C# program that asks the user to provide five pieces of personal information.

Design a class where:

* All attributes are encapsulated
* Data is accessed securely using getters and setters

In the main form:

* Add textboxes for user input
* Add a button that displays all entered information inside a MessageBox

The application should demonstrate:

* Proper use of encapsulation
* Clean program structure
* User interaction through GUI
